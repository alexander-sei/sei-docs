<Danger>
  **Deprecation Notice**

  Cosmos SDK and CosmWasm functionality will be deprecated in favor of EVM-only. For more details, see [SIP-3](https://github.com/sei-protocol/sips/blob/main/sips/sip-3.md) and the [governance proposal](https://www.mintscan.io/sei/proposals/99).
</Danger>

## Cosmos-SDK type Endpoints

See [EVM Networks](/evm/networks) for current chain IDs and RPC endpoints. Cosmos-SDK RPC access remains available on the same hosts under standard Tendermint ports.
