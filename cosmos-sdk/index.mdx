<Warning>
  **Deprecation notice**

  Cosmos SDK and CosmWasm functionality will be deprecated in favor of EVM-only. See [SIP-3](https://github.com/sei-protocol/sips/blob/main/sips/sip-3.md) and the [governance proposal](https://www.mintscan.io/sei/proposals/99) for details.

  Developers should [migrate to the EVM docs](/evm) for all new work. See the [SIP-03 migration guide](/learn/sip-03-migration) for affected assets and next steps.
</Warning>

## Cosmos-SDK endpoints

| Network  | Chain ID    | REST                               | gRPC                              |
| -------- | ----------- | ---------------------------------- | --------------------------------- |
| Mainnet  | pacific-1   | `https://rest.sei-apis.com`        | `https://grpc.sei-apis.com`       |
| Testnet  | atlantic-2  | `https://rest-testnet.sei-apis.com` | `https://grpc-testnet.sei-apis.com` |

<Info>
  EVM precompiles remain the supported path for accessing Cosmos-SDK modules (bank, staking, governance, distribution, etc.). See [EVM Precompiles](/evm/precompiles/example-usage) and the [CosmWasm precompile bridge](/evm/precompiles/cosmwasm-precompiles/example-usage).
</Info>
