Connect your wallets, dApps, and nodes to Sei using the values below. For production workloads, prefer a [dedicated RPC provider](/learn/rpc-providers) over the public endpoints.

## Mainnet — Pacific-1

| Property        | Value                                                   |
| --------------- | ------------------------------------------------------- |
| Chain ID        | `1329` (`0x531`)                                        |
| Currency        | SEI                                                     |
| EVM RPC         | `https://evm-rpc.sei-apis.com`                          |
| EVM WebSocket   | `wss://evm-ws.sei-apis.com`                             |
| REST (Cosmos)   | `https://rest.sei-apis.com`                             |
| gRPC (Cosmos)   | `https://grpc.sei-apis.com`                             |
| Explorer        | [seitrace.com](https://seitrace.com)                    |

<Info>
  All endpoints are public and rate-limited. For production apps, use a [dedicated RPC provider](/learn/rpc-providers) or [run your own node](/node).
</Info>

## Add Sei to your wallet

Most EVM wallets (MetaMask, Rabby, Frame, Trust) accept a chain config via [chainlist.org/chain/1329](https://chainlist.org/chain/1329). Or add the network manually using the values above.
